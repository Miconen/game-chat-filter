# game-chat-filter
List of regex based chat message filters for Oldschool Runescape to be used with the "[Chat Filter Updater](https://github.com/IamReallyOverrated/Runelite_ChatFilter)" RuneLite plugin.

## How to use?
1. Install the "Chat Filter Updater" plugin from the plugin hub.
2. Navigate to the settings and replace the "Filter URL" field with `https://raw.githubusercontent.com/Miconen/game-chat-filter/refs/heads/main/filter`
3. Make sure to enable the "Filter game chat" option from the "Chat Filter" plugin.
4. You might have to restart your RuneLite client for the filter url to take effect.
